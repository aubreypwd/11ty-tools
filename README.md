# `11ty-starter`

`11ty-starter` is my personal starter project for building websites with 11ty. It's highly opinionated and is intended only for my personal use, so **use at your own risk.**

# `11ty-starter-common`

`11ty-starter-common` is a repo of shared functionality that can be updated to make all `11ty-starter` websites better. It is a `git subtree`. See `package.json::scripts` where you will find `tools::` commands for contributing upstream and pulling.
